# FIST FURY 3D — 2本指で戦う3Dボクシングゲーム

Three.js 製の一人称3Dボクシングゲーム。スマホの2本指ジェスチャーで左右の腕を操作し、
CPU戦とオンライン対戦(合言葉ルーム)ができます。

## 遊び方

### 起動

```bash
npm install
npm start
```

ブラウザで <http://localhost:8787> を開く。
オンライン対戦する場合は、同じサーバーに全員がアクセスすればOK
(LAN内なら `http://<このPCのIP>:8787`。インターネット越しなら Render / Railway / fly.io などに
このリポジトリをそのままデプロイして、そのURLを共有)。

CPU戦だけなら `index.html` を静的サーバー(GitHub Pages等)で配信するだけでも遊べます。

### 操作(画面の左半分=左腕、右半分=右腕)

| 操作 | 技 |
| --- | --- |
| タップ | ストレート(顔面) |
| 内側へのサイドスワイプ | 相手の反対側の半身へのボディブロー |
| 真ん中から上へスワイプ | 顔面へのアッパー |
| 下から真ん中への上フリック | フック |
| 長押し(押している間) | ガード。腕のような縦長の角丸矩形が指に追従 |

- 相手のパンチは着弾点に赤いリングで予告される。そこにガード矩形を重ねればブロック成功。
- ブロックされた側は **0.2秒間、攻撃もガードもできない**。
- 被弾すると画面の端が赤く光る。

### KO
HPが0になるとダウン。KO率は最初5%で、**試合時間の経過とダウン回数で上昇**していく。
KOを免れれば3秒後に立ち上がる(回復量はダウンを重ねるほど減る)。4回ダウンでTKO。

### オンライン対戦
1. 初回起動時にリングネームを登録(接続中の名前と重複するとサーバーに拒否される)
2. 「オンライン対戦」→ サーバーURLと**部屋の合言葉**を入れて入室
3. 部屋に**二人だけなら自動で試合開始**。三人以上いる場合はメンバー一覧から対戦申請 → 相手が承認で開始

### 顔ペイント
自分のキャラクターの顔はペイントキャンバスで自由に描けて、3Dモデルの頭に貼られます。
オンライン対戦では相手にも自分の顔が見えます。

## 構成

| ファイル | 内容 |
| --- | --- |
| `index.html` / `css/style.css` | 画面・HUD・エフェクト |
| `js/gesture.js` | 2本指ジェスチャー判定とガード矩形 |
| `js/boxer.js` | 3D人型モデル。2ボーンIKでパンチ軌道どおりに腕が動く |
| `js/game.js` | 試合ロジック(ダメージ・ガード・スタン・ダウン・KO率・演出) |
| `js/ai.js` | CPUの思考(攻撃選択・反応ガード) |
| `js/net.js` / `server.js` | オンライン対戦(名前重複禁止・合言葉ルーム・自動マッチ・中継) |
| `js/facepaint.js` | 顔ペイント |
| `vendor/three.min.js` | Three.js r128(同梱) |
